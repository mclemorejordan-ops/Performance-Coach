Permissions Troubleshooting

  • If outdoor cardio tracking is not working, verify Location access is allowed while using the app.
  • If pace or movement detection is limited, verify Motion access is enabled.
  • If Apple Health writes are unavailable, verify Health permissions for Performance Coach.

Account Deletion

If you created an account in the app, you can delete it in-app from the Settings support section.
